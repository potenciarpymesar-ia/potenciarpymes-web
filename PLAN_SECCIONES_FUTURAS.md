# Potenciar Pymes — Secciones futuras recomendadas

Estas secciones conviene agregarlas de forma progresiva, no todas juntas.

## Prioridad 1

### 1. Diagnóstico más fuerte
Página o bloque que explique:
- qué mira el diagnóstico;
- qué recibe el usuario;
- cuánto tarda;
- qué pasa después;
- ejemplos de problemas detectados.

### 2. Servicios con páginas propias
Crear primero:
- `/servicios/tiendanube`
- `/servicios/mercado-libre`
- `/servicios/seo-geo`
- `/servicios/automatizaciones`

Cada página debe tener CTA al diagnóstico.

### 3. Casos reales con evidencia
No usar claims fuertes sin prueba.
Formato recomendado:
- problema inicial;
- qué se hizo;
- qué cambió;
- captura o evidencia si existe;
- aprendizaje.

## Prioridad 2

### 4. Recursos descargables
Primer recurso recomendado:

> Checklist: ¿tu pyme tiene sistema digital o solo herramientas sueltas?

Sirve para captar leads y nutrir WhatsApp/email.

### 5. Google Business / SEO local
Bloque o página:
- Potenciar Pymes en Buenos Aires;
- servicios para pymes argentinas;
- diagnóstico remoto o AMBA;
- preguntas frecuentes locales.

### 6. Automatizaciones
Mostrar ejemplos simples:
- formulario → WhatsApp;
- consulta → etiqueta;
- lead → seguimiento;
- Tiendanube/ML → tablero;
- reporte mensual.

## Prioridad 3

### 7. Página “Método”
Explicar el sistema:
1. Diagnóstico.
2. Orden.
3. Implementación.
4. Medición.
5. Optimización.

### 8. Página “Herramientas”
No vender herramientas como protagonista. Mostrar para qué se usan:
- Tiendanube para tienda propia;
- Mercado Libre para demanda activa;
- WhatsApp para conversación;
- GA4 para medición;
- IA para ahorrar tiempo;
- automatizaciones para procesos repetidos.

### 9. Comparativas SEO/GEO
Recursos:
- Tiendanube o Mercado Libre: cuál conviene.
- Qué medir en una pyme digital.
- Qué automatizar primero.
- Cómo saber si tus anuncios pierden plata.

## Regla

No agregar secciones por agregar. Cada sección debe cumplir una función:

- aclarar;
- generar confianza;
- resolver objeción;
- mejorar SEO;
- aumentar conversión;
- llevar al diagnóstico.
