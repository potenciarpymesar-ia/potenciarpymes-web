# Prompt para Claude — Fase 2 Conversión Potenciar Pymes

Trabajá sobre el repositorio de Potenciar Pymes.

Ya existe una Fase 1 técnica con SEO base, redirects, robots, sitemap, llms, schema y eventos.

Ahora implementá la Fase 2 de conversión usando los archivos de esta carpeta como referencia.

## Objetivo

Mejorar la claridad comercial de la home y del diagnóstico sin hacer todavía un rediseño visual extremo.

## Instrucciones

1. Compará `index.html` y `diagnostico.html` de esta Fase 2 contra el repo actual.
2. Implementá los cambios de copy y estructura.
3. Conservá el diseño actual salvo que haya un problema claro de responsive, accesibilidad o jerarquía.
4. Agregá la sección `Qué te llevás del diagnóstico`.
5. No agregues animaciones pesadas.
6. No conviertas la web en una plantilla SaaS genérica.
7. No uses promesas de resultados que no estén probadas.
8. No agregues métricas duras si no están verificadas.
9. Mantené español argentino, simple y directo.
10. Ejecutá una revisión responsive mobile.

## Criterio visual

La web debe verse:

- moderna;
- clara;
- profesional;
- simple;
- confiable;
- orientada a pymes argentinas.

No debe verse:

- futurista;
- gamer;
- demasiado tech;
- genérica de agencia;
- sobrecargada de efectos.

## Archivos a revisar

- `index.html`
- `diagnostico.html`

## Validaciones

Antes de cerrar, verificá:

- un solo H1 por página;
- CTAs funcionando;
- `/diagnostico` funcionando;
- `/diagnostico.html` redireccionando correctamente si corresponde;
- formulario Netlify funcionando;
- eventos no rompen la página si GA4/GTM no está instalado;
- mobile usable;
- textos sin promesas exageradas;
- build/deploy funcionando.

## Entregable

Devolvé:

1. resumen de cambios;
2. archivos modificados;
3. qué secciones agregaste;
4. qué no pudiste verificar;
5. recomendaciones visuales para Fase 3.
